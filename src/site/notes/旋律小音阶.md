# 旋律小音阶

旋律小音阶在和声小音阶的基础上，在上行旋律中将第六个音升高半个音，这样第六个音和第七个音便不是增二度关系，听起来会更顺耳。在下行旋律中，第七个音不用接到主音，所以第七个音不需要升高半音，连带着第六音也不需要升高了。
