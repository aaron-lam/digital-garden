感覺統合失調會使寶寶對外界的刺激產生比較極端的反應。治療方法如下：

# 1. 觸覺訓練

用不同材質的毛巾或者手去撫摸寶寶的身體，例如洗澡後。也可以給寶寶不同材質的玩具，比如布書、毛絨玩具等。

# 2. 聽覺訓練

播放風琴樂曲、古典音樂或者寶寶童謠。

# 3. 視覺訓練

把各種各樣的繪本讀給孩子聽。

# 4. 平衡感

玩“寶寶飛“。
