# 旋律小音階

旋律小音階在和聲小音階的基礎上，在上行旋律中將第六個音升高半個音，這樣第六個音和第七個音便不是增二度關係，聽起來會更順耳。 在下行旋律中，第七個音不用接到主音，所以第七個音不需要升半音，連帶第六個音也不需要再升高了。
