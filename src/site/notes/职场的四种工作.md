# T1: 形象工程
高层们都关心的重点项目，且需要付出的劳动代价极低。这种优质项目一般只给猪和狐狸做。

# T2: 日常工作
普通的工作繁琐且用处不大。一般由蠢驴承担。

# T3: 垃圾工作
麻烦且不讨喜的工作，比如负责汇报坏消息给上司。蠢驴不会分辨这种工作，故这种工作都会推给他。

# T4: 陷阱工作
能力再大也做不好的工作，且项目失败了有可能会解雇你。
