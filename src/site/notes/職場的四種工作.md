# T1: 形象工程
高層們都關心的重點項目，且需要付出的勞動代價極低。這種優質項目一般只給豬和狐狸做。

# T2: 日常工作
普通的工作繁瑣且用處不大。一般由蠢驢承擔。

# T3: 垃圾工作
麻煩且不討喜的工作，比如負責匯報壞消息給上司。蠢驢不會分辨這種工作，故這種工作都會推給他。

# T4: 陷阱工作
能力再大也做不好的工作，且項目失敗了有可能會解僱你。
